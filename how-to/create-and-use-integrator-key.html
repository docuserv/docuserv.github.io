Create & Use your DocuSign Integrator Key

Create your DocuSign Integrator Key

Please follow the steps below to create an integrator key in your Developer account. If you don't have a Demo account, please sign up for a free developer account, and you will be sent an activation email message.

Once you have activated your Developer account:

  • Log into your account, via https://demo.docusign.net/
  • Select the option to "Go to Admin" from the dropdown menu by your profile image (top-right)
  • Under INTEGRATIONS, select the "API and Keys" option
  • Click the ADD INTEGRATOR KEY button
  • Enter an App Description for your integration, then click on the ADD button
  • Copy and Paste into a text editor the following items from the page:
    • API Username
    • Integrator Key

Prepare your X-DocuSign-Authentication header value

The X-DocuSign-Authentication header is used as part of any HTTP(S) request to the DocuSign Web Service APIs, and incorporates:

  • Your username (recommended to use the API Username seen when creating your DocuSign integrator key)
  • Your password (recommended to use the API Password - see Get your DocuSign API Password)
  • Your integrator key

The value for the X-DocuSign-Authentication header can be specified in XML or JSON format:

JSON: {"Username":"username", "Password":"password", "IntegratorKey":"integratorKey"}

XML: <DocuSignCredentials><Username>username</Username><Password>password</Password><IntegratorKey>integratorKey</IntegratorKey></DocuSignCredentials>
