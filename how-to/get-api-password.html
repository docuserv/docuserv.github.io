Get your DocuSign API Password

If you have not already done so, register for a free DocuSign Developer account at https://secure.docusign.com/signup/developer, and you will be sent an activation email message.

Once you have activated your Developer account:

  • Navigate to DocuSign's API Explorer at https://apiexplorer.docusign.com/
  • Click the button to "Authenticate using Sandbox Account"
  • A dialog should appear, asking for your OAuth Access Token
  • If you have an Access Token, you can enter it here, otherwise click on the "Get OAuth2 Token" button
  • A pop-up window should appear, prompting you to enter your username (Email address)
  • Followed by your password
  • Your main browser window should refresh upon successful authentication, to show:
  • In this window, choose "true" from the "api_password" drop-down, then click the "Send Request" button
  • After a second, when you scroll down the page you should see a "Response" section, and within this a "Data" section
  • Within the "Data" section is your API Password (with surrounding double quotes):
